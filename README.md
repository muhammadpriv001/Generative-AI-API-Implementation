# Generative-AI-API-Implementation
Using generative AI API --- Giving visual and textual data to a chatbot at once? It's possible and can be made efficient, you can tweak your chatbot according to your needs.

## Setup:
1. You will need an API key of Gemini-ai. you can have it from website for free.
2. Create a .env file in the project directory and write it like:
          API_KEY = YOUR_API_KEY
3. Run _init_.py once

### You are Ready to Go!
